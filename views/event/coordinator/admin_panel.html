NOTICE

  1. Event co-ordinators need to check all the participants in before allowing them in their event.
  2. In order to check a participant in the event, visit https://innonitr.com/eventauth and authorize by providing the corresponding loginID and password of the event.
  3. On successful authorization, a message 'Authorized' will appear on the browser. Else, it will show an error message regarding misauthorization.
  4. So, after a successful login, keep the browser open and intact in the background. This will authorize you for QR code scanning.
  5. Use any of the QR code scanner to scan the QR code of the participants from their profile.
  6. After scanning, if message 'Successfully checked-in' appears, then the participant gets checked-in into the event and then is eligible to participate for the event.
  7. If the message 'User hasn't checked-in into the institute' appears, then either user hasn't checked-in into the institute or he hasn't paid yet.
  8. If the participant hasn't registered for the event, then an error 'User has not registered for the event.' will appear. Please ask them to register to your event first in website.
  9. In any other cases please try again.
